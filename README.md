# Mobile Sales Dashboard - Power BI

## Project Overview
This project is an interactive Mobile Sales Dashboard developed using Power BI to analyze sales performance, customer trends, and brand-wise insights.

## Features
- Total Sales Analysis
- Monthly Revenue Trends
- Brand-wise Performance
- Customer Purchase Insights
- Region-wise Sales Distribution
- Interactive Filters & Slicers

## Tools & Technologies
- Power BI
- DAX
- Excel
- Data Visualization

## Key Insights
- Identified top-selling mobile brands
- Analyzed monthly sales growth
- Compared regional performance
- Tracked customer purchasing behavior
